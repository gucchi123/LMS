# PythonAnywhere デプロイチェックリスト

LMSをPythonAnywhereにデプロイする際のチェックリストです。

## 📋 デプロイ前

- [ ] GitHubリポジトリを作成
- [ ] ローカルでLMSが正常に動作することを確認
- [ ] `.gitignore` に大きな動画ファイルが含まれていることを確認
- [ ] コードをGitHubにプッシュ

## 🔧 PythonAnywhere設定

### アカウント
- [ ] PythonAnywhereアカウント作成（無料）
- [ ] メールアドレス確認完了

### コードのアップロード
- [ ] Bashコンソールを開いた
- [ ] `git clone` でコードをダウンロード
- [ ] LMSディレクトリに移動 (`cd ~/LMS`)

### Python環境
- [ ] 仮想環境を作成 (`mkvirtualenv --python=/usr/bin/python3.10 lms-env`)
- [ ] パッケージをインストール (`pip install -r requirements.txt`)
- [ ] データベースを初期化 (`python init_db.py`)

### Webアプリ設定
- [ ] Webアプリを作成（Manual configuration, Python 3.10）
- [ ] 仮想環境のパスを設定 (`/home/YOUR_USERNAME/.virtualenvs/lms-env`)
- [ ] WSGIファイルを編集
  - [ ] `YOUR_USERNAME` を実際のユーザー名に変更
  - [ ] `SECRET_KEY` を強力なランダム文字列に変更
  - [ ] 保存
- [ ] 静的ファイルを設定
  - [ ] URL: `/videos`
  - [ ] Path: `/home/YOUR_USERNAME/LMS/videos`

### 起動
- [ ] 「Reload」ボタンをクリック
- [ ] 公開URLにアクセス (`https://yourusername.pythonanywhere.com`)
- [ ] ログイン画面が表示されることを確認

## 🔐 セキュリティ

- [ ] `admin` アカウントでログイン成功
- [ ] 管理画面にアクセス可能
- [ ] **デフォルトパスワードを変更** (重要！)
- [ ] 新しいユーザーを作成してテスト

## 🎬 動画テスト

- [ ] 管理画面から小さな動画をアップロード（テスト用）
- [ ] 動画が正常に再生されることを確認
- [ ] 視聴進捗が保存されることを確認
- [ ] 続きから再生できることを確認

## 📊 動作確認

- [ ] ログイン/ログアウトが正常に動作
- [ ] ダッシュボードで動画一覧が表示される
- [ ] 動画の視聴が正常に動作
- [ ] 進捗が保存される
- [ ] 管理者機能が正常に動作
  - [ ] 動画アップロード
  - [ ] 動画編集
  - [ ] 動画削除
  - [ ] ユーザー統計表示

## 🐛 トラブルシューティング

エラーが発生した場合：

- [ ] Error log を確認 (「Web」タブ → 「Error log」)
- [ ] Server log を確認
- [ ] パスが正しいか確認（ユーザー名のタイプミスなど）
- [ ] 仮想環境が正しく設定されているか確認
- [ ] `DEPLOY_PYTHONANYWHERE.md` のトラブルシューティングセクションを参照

## 📢 公開

すべて確認できたら：

- [ ] 公開URLをチームに共有
- [ ] ログイン情報を共有（新しいパスワード）
- [ ] 使い方を説明（`QUICKSTART.md` を共有）

## 🎉 完了！

すべてのチェックが完了したら、LMSの公開は成功です！

公開URL: `https://yourusername.pythonanywhere.com`
